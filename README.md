# Daily English News Reader

A free daily English learning news website built with Next.js App Router, TypeScript, and Tailwind CSS.

## Features

- Landing page with a large greeting that changes every 5 seconds
- Interface language selector saved to localStorage
- English level selector saved to localStorage
- Mock news listing page
- Article reading practice pages
- English article text with localized UI, instructions, and vocabulary meanings
- No paid APIs, no AI APIs, and no paid database

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
